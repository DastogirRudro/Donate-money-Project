What is the Document Object Model(DOM)?

It is a programming interface for web documents which represents the structure of a webpage

To select an element from the DOM (Document Object Model),we can use various JavaScript methods depending on how you want to select the element.

What is event delegation in the context of the DOM,and why is it useful?

It is a technique in the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements.It is very useful for various reason
